## 1. Introduction
The purpose of this test plan is to validate the functionality, usability, and reliability of the CultureCheck web application. The app is used to submit and view simple "stories" (title + description) and is designed as a QA portfolio project.

## 2. Scope

### In Scope
- UI testing (React)
- Form validation
- Story creation workflow
- Story deletion
- Regression testing

### Out of Scope
- Authentication
- API backend
- Multi-user support

## 3. Test Types
- Functional testing
- Negative testing
- Boundary testing
- Regression testing
- Usability testing

## 4. Environment
- Local: http://localhost:5173
- OS: Windows 11
- Browser: Chrome (latest)

## 5. Entry Criteria
- App builds successfully.
- No blocking UI issues.

## 6. Exit Criteria
- All high-priority test cases pass.
- No critical defects open.
